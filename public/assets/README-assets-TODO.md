# Assets — generated via Pillow (gen_images.py)

These files are generated by `gen_images.py` at the repo root using Pillow.
All images now use the dark + orange design system palette.

## Palette
- Background: `#0B0B0D`
- Orange accent: `#FF8A4C` → `#FF6B3D` (gradient)
- Muted text: `#B8B8BD`
- White: `#FFFFFF`

## og-image.png (1200×630)
Dark `#0B0B0D` background, orange glow blobs, "Merge" wordmark in white Lato-Bold,
tagline in muted (`#B8B8BD`), orange gradient accent bar at top.

## favicon.ico
"M" on orange gradient rounded square (`#FF8A4C`→`#FF6B3D`), multi-size (16, 32, 48).

## apple-touch-icon.png (180×180)
Same "M" on orange gradient rounded square at 180×180.

Run `python3 gen_images.py` from the repo root to regenerate all three.
